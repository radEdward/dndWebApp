Welcome to an easy-to-use virtual table top for your role playing game needs. Click anywhere to make this window disappear.

I made this as a virtual game board to be displayed on a large monitor/television for your players while you run the game. Get started with a random test map by clicking Random Test Map above. If you have a map image you'd like to use, click Choose File and get to the game.

Click on the next page to get a tutorial of the controls

This was made taking 100 pixels to be equivalent to 5 feet in game, so make sure your map is scaled appropriately.

The icon images are a part of Devin Night's free token pack. The test maps are from Dyson's Dodecahedron and from donjon's random dungeon generator. The wood background is vector art by vecteezy.com
